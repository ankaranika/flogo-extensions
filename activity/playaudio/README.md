
# Play Audio - Activity

## Installation

```bash
flogo install github.com/ankaranika/flogo-extensions/activity/playaudio
```
Link for flogo web:
```bash
https://github.com/ankaranika/flogo-extensions/activity/playaudio
```

## Schema
Inputs and Outputs:

```json
{
  "inputs":[
    {
      "name": "audio",
      "type": "any"
    }
  ]
}
```
## Inputs
| Input   | Description    |
|:----------|:---------------|
| audio | The audio to be played |
