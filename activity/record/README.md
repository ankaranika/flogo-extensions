
# Record - Activity

## Installation

```bash
flogo install github.com/ankaranika/flogo-extensions/activity/record
```
Link for flogo web:
```bash
https://github.com/ankaranika/flogo-extensions/activity/record
```

## Schema
Inputs and Outputs:

```json
{
  "outputs":[
    {
      "name": "recording",
      "type": "any"
    }
  ]
}
```
## Outputs
| Output   | Description    |
|:----------|:---------------|
| recording | The recorded audio |
