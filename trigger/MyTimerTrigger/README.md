
# MyTimerTrigger - Trigger

## Installation

```bash
flogo install github.com/ankaranika/flogo-extensions/tigger/MyTimerTrigger
```
Link for flogo web:
```bash
https://github.com/ankaranika/flogo-extensions/tigger/MyTimerTrigger
```
